# Open BrandKit Website

Marketing site for [Open BrandKit](https://github.com/campbell17/open-brandkit).

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connected to Vercel. Pushes to `main` deploy to production.

The product package lives in the separate [`open-brandkit`](https://github.com/campbell17/open-brandkit) repository.
